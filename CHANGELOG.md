# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-03

### Added
- Initial release of OneView framework
- Core application and container system
- Reactive view system with state management
- Client-side router with route management
- View lifecycle hooks (onMount, onUnmount, onUpdate)
- ViewManager and ViewLoader for dynamic view loading
- ViewState and StateManager for reactive updates
- HTTP service (AppAPI) for API calls
- Event service for event management
- Storage service for local/session storage
- DOM utilities (OneDOM) for DOM manipulation
- Template engine (OneMarkup) with variable interpolation
- Helper utilities for common tasks
- SSR (Server-Side Rendering) support with data parsing
- TypeScript support with full type safety
- Build system with TypeScript compilation
- CLI tools for building views
- Watch mode for development
- Type declarations and source maps
- Module exports for tree-shaking

### Features
- **Reactive Views**: Automatic UI updates when state changes
- **TypeScript First**: Full TypeScript support with strict mode
- **Modular Architecture**: Component-based design pattern
- **Laravel Integration**: Seamless integration with Laravel backends
- **Performance**: Lightweight bundle with tree-shaking support
- **Developer Experience**: Excellent DX with watch mode and hot reload
